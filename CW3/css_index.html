1) Best way to make this possible is by having multiple css style sheets for each department. Then for each teacher they will want to follow the style of the department page but could have some liberty into some minor aspects like format of exam times or time tables by using in-line styles

2) First we'll want to make the homepage which will be the basis for how all the other webpages will be formatted, so it's best to use and external css file that can be edited to make changes to all webpages if need. Then, for each department they can make another css file that follows the homepage but is slightly different for the sake of uniqueness. For each teacher's webpage they can either use internal style sheets or inline style sheets to achieve their goal. Or their department's external style sheet could have various different options for the same thing, so like having multiple css classes for making a table. This is better because the teachers won't have the full liberty in making their webpage drastically different from the school's/department's structure, but it allows them some freedom to make it their own and adding what works for them. It's basically giving them a set of options to choose from.
